Тема: Блочная модель и Flexbox

📐 1. Блочная модель в CSS

Каждый HTML-элемент — это прямоугольник. Блочная модель описывает, как рассчитывается его размер с учётом: content (содержимое), padding (внутренний отступ), border (рамка), margin (внешний отступ).

🎯 Практический пример

Я — блок

Разбор:

  • width: 200px; height: 100px; — размер содержимого
  • padding: 20px; — добавляет 20px отступов внутри со всех сторон
  • border: 5px; — добавляет рамку толщиной 5px
  • margin: 30px; — создаёт отступ в 30px от соседних элементов
  • Итоговая ширина блока = 200 + 20*2 + 5*2 = 250px (без учёта margin)

💡 Свойство box-sizing: border-box упрощает расчёты, заставляя width/height включать padding и border.

🧩 2. Изучение Flexbox на практике

Flexbox — это технология для удобного выравнивания и распределения элементов внутри контейнера.

1️⃣ display: flex

Контейнер становится flex-контейнером, а его дочерние элементы — flex-элементами.

📌 Шпаргалка по Flexbox (для себя)

  • display: flex; — включает Flexbox
  • flex-direction: row | column | row-reverse | column-reverse — направление оси
  • justify-content: flex-start | center | space-between | space-around | space-evenly — выравнивание по главной оси
  • align-items: stretch | center | flex-start | flex-end — выравнивание по поперечной оси
  • flex-wrap: wrap | nowrap — перенос строк
  • gap: 10px; — расстояние между элементами
  • flex-grow: число — растяжение элемента
  • order: число — порядок отображения
